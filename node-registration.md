# 2.3.3 边缘设备注册

Cantor 提供全面的边缘设备和 IoT 节点注册功能，使其能够无缝集成到集群中。

- **集成**:
  - Cantor 集成了 **Vega**（云端或本地注册中心），用于管理大规模边缘和 IoT 设备。
  - 注册中心支持动态查询设备状态和配置管理。

- **租户隔离注册**:
  - 每个客户通过 **租户隔离注册** 实现数据和设备隔离。
  - 使用公钥/私钥认证机制简化注册流程并提升安全性。

- **令牌验证**:
  - 注册中心通过 **智能时间窗口** 验证令牌，确保未授权设备无法访问集群。
  - 确保设备在严格认证条件下参与集群。
